Vinduer Røros Isoler Sidehengslet/Topphengslet

Sidehengslet/Topphengslet

Sidehengslet med profilert midtpost
Sidehengslet med glatt profil

Profiler

Rørosvinduet Isoler Sidehengslet leveres med profilert midtpost. Kan også leveres med glatt profil. Kan også leveres med midtpost som tilfredsstiller krav til rømningsvei (løs midtpost).

Standard vrider
Haspe 175 i gull og krom
Haspe 175 i sort og hvit
Haspe 1324 (dråpehaspe) i messing
Haspe 1324 i krom
Haspe MP380 i messing

Vridere og hasper

Røros Isoler Sidehengslet leveres standard med blank vrider. Kan bestilles med hasper i ulike utførelser og farger.

Hengsel blank
Hengsel hvit
Hengsel blank med pynteknopp
Hengsel sort med pynteknopp
Hengsel rustfri med pynteknopp

Hengsel

Hengsel leveres standard i elforsinket blank uten knopp. Fargemuligheter: Blank, hvit og sort. Kan også leveres med pynteknopp på forespørsel. Fargemuligheter: Blank,hvit og sort. I tillegg kan hengsler leveres i rustfri utførelse.

Standard stormkrok
S-stormkrok i gull, sort og hvit
Sikkerhetsstormkrok

Stormkrok

Vi leverer stormkrok som el forsinket. Stormkroker kan også leveres i en annen utførelse og fargeutvalg. Stormkroker kan også leveres som sikkerhetsstormkrok i hvit eller gulkrom.

Ventiler

Røros Vinduet Isoler leveres uten ventil som standard. Kan leveres på forespørsel.

Sprosser

- Vedlikeholdsfrie utenpåliggende aluminiums sprosser i 26, 45 og 60 mm bredde
- Utenpåliggende tresprosser med klips (26 og 61 mm)
- Gjennomgående tresprosser 58 mm
- Wienersprosse
- Kittsprosse 29 mm
- Alternative sprosseløsninger på forespørsel

Glassvalg

Standard glass er 2-lags energiglass med varmespacer. Vinduene kan levere med en rekke alternative glassvalg som for eksempel.:. Sikkerhetsglass, Støyglass, selvrensende, laminert glass for beskyttelse av uv- stråler, sol-og varmeregulerende glass osv. i 2- og 3-lags løsninger. Foringer Vinduene kan leveres med tilpassede foringer om ønskelig.
